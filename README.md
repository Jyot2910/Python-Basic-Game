🎮 Python Basic Games

This project contains simple and fun beginner-level games built using Python. It includes classic games like Snake Game and Tic Tac Toe, designed to help understand basic programming concepts, logic building, and game development fundamentals.

📌 Games Included
🐍 Snake Game
Classic snake game where the snake grows after eating food
Avoid collisions with walls and itself
Score increases as the snake eats food
❌⭕ Tic Tac Toe
Two-player game (Player X vs Player O)
Simple grid-based gameplay (3x3)
Detects winner or draw automatically
🛠️ Technologies Used
Python 3
pygame (for Snake Game)
Basic Python (for Tic Tac Toe)
